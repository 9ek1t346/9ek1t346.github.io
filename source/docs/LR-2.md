    # Лабораторная работа №2

## Тема
Основы NumPy: массивы и векторные операции.

## Цель работы
Изучить базовые возможности библиотеки NumPy для работы с одномерными и двумерными массивами, освоить векторные и матричные операции, методы статистического анализа данных и построение простых графиков.

## Постановка задачи
В рамках лабораторной работы необходимо реализовать набор функций для:
- создания и преобразования массивов;
- выполнения векторных операций;
- выполнения матричных операций;
- загрузки и анализа данных из CSV-файла;
- построения графиков и тепловой карты.

Также код должен проходить встроенные тесты и быть оформлен с учетом требований к стилю.  

## Структура проекта
В задании предложена следующая структура проекта:
- `main.py`
- `test.py`
- `data/students_scores.csv`
- `plots/`

## Что требовалось реализовать

### 1. Создание и обработка массивов
В первой части лабораторной работы требовалось:
- создать вектор от 0 до 9;
- создать случайную матрицу 5x5;
- изменить форму массива;
- выполнить транспонирование матрицы.

### 2. Векторные операции
Во второй части требовалось реализовать:
- сложение векторов;
- умножение вектора на скаляр;
- поэлементное умножение;
- скалярное произведение.

### 3. Матричные операции
В третьей части лабораторной работы необходимо было реализовать:
- умножение матриц;
- вычисление определителя;
- нахождение обратной матрицы;
- решение системы линейных уравнений.

### 4. Статистический анализ
Отдельный блок был посвящен работе с данными:
- загрузке CSV-файла;
- вычислению среднего значения;
- вычислению медианы;
- нахождению стандартного отклонения;
- поиску минимума, максимума и квартилей;
- нормализации данных по формуле Min-Max.

### 5. Визуализация
В последней части работы нужно было:
- построить гистограмму;
- построить тепловую карту корреляции;
- построить линейный график.

## Подход к решению
При выполнении лабораторной работы основное внимание уделялось использованию векторизированных возможностей NumPy без лишних циклов.  
Для линейной алгебры применяются функции `numpy.linalg`, для загрузки CSV — `pandas`, а для визуализации — `matplotlib` и `seaborn`.

Такой подход позволяет писать более компактный, понятный и быстрый код по сравнению с ручной реализацией операций через обычные списки Python.

## Нюансы реализации
При выполнении задания важно учитывать несколько моментов:
- векторные операции лучше выполнять средствами NumPy без циклов;
- для матричных операций нужно следить за размерностями массивов;
- при нахождении обратной матрицы и решении системы уравнений матрица должна быть невырожденной;
- при нормализации данных нужно учитывать случай, когда все значения одинаковые;
- графики следует сохранять в каталог `plots`, чтобы результаты работы можно было проверить отдельно.

## Использованные инструменты
В файле задания указаны следующие зависимости:
- `numpy`
- `matplotlib`
- `seaborn`
- `pandas`
- `pytest`

## Результат
В ходе лабораторной работы подготавливается модуль с реализацией функций NumPy, который должен:
- корректно выполнять операции над массивами и матрицами;
- выполнять статистический анализ данных;
- строить и сохранять графики;
- успешно проходить встроенные тесты.

Работа формирует базу для дальнейшего использования NumPy и связанных библиотек при решении задач анализа данных.

Статический сайт-портфолио, созданный в рамках лабораторной работы №1 с использованием MkDocs и публикацией через GitHub Pages.

## Ссылки
- Репозиторий: https://github.com/9ek1t346/9ek1t346.github.io
- Сайт: https://9ek1t346.github.io/

## Выбор темы оформления
Для сайта выбрана тема **Material for MkDocs**.

Причины выбора:
- современный и аккуратный внешний вид;
- удобная иерархическая навигация;
- хорошая читаемость на компьютере и телефоне;
- встроенный поиск и удобное меню;
- подходит для оформления учебного портфолио и документации.
